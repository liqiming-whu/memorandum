## 安装中文支持包language-pack-zh-hans

```shell
sudo apt-get install language-pack-zh-hans
```

## 修改/etc/environment

```shell
sudo vim /etc/environment
```

在文件末尾追加：


```shell
LANG="zh_CN.UTF-8"
LANGUAGE="zh_CN:zh:en_US:en"
```

## 修改/var/lib/locales/supported.d/local

创建local文件

```shell
sudo vim /var/lib/locales/supported.d/local
```

在文件中加入


```shell
en_US.UTF-8 UTF-8
zh_CN.UTF-8 UTF-8
zh_CN.GBK GBK
zh_CN GB2312
```

## 执行

```shell
sudo locale-gen
```
